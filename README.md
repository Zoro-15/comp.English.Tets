# English Mock Tests

A modern, high-performance, and mobile-friendly mock examination platform designed for competitive English exams.

The site is hosted and available online.

## Features

- **Progressive Test Recommendations:** Dynamically tracks student test history to offer the next recommended mock exam.
- **Interactive Timer:** 20-minute timed simulation that automatically submits and grades the test.
- **Smart Navigation Grid:** A visually responsive overview of all questions with status indicators (answered, current, unanswered).
- **Comprehensive Analytics Dashboard:** Detailed test reports displaying score, accuracy percentage, time taken, and correct/incorrect breakdown.
- **Detailed Solution Explanations:** Post-test review with complete answer analysis and explanations for every question.
- **Responsive Layout:** A premium, rounded aesthetic that adapts perfectly to desktop and mobile screens.

## Tech Stack

- **Astro** - Routing and server-side loading container
- **React** - Interactive component framework
- **Tailwind CSS v4** - Custom styled modern UI design
- **Supabase** - Live Postgres backend database
